Features

Everything you need for Nepali date handling in your web applications

Date Conversion

Convert dates between Bikram Sambat (BS) and Gregorian (AD) calendars with high accuracy and support for a wide range of years.

Date Picker UI

Ready-to-use Nepali date picker component with customizable themes, formats, and language support for seamless integration.

Multilingual Support

Full support for both English and Nepali (Devanagari) languages with proper digit localization and month names.

TypeScript Ready

Built with TypeScript with full type definitions, providing excellent developer experience and code completion.

Customizable Themes

Light and dark themes with easy customization options to match your application's design system.

Flexible Formatting

Powerful date formatting with custom format strings, supporting various display patterns and localization options.

Installation

Get started with NepaliPatroJS in your project

NPM

Install via npm for Node.js projects:

npm install nepalipatrojs

Then import:

import NepaliPatro from 'nepalipatrojs';

Yarn

Install via Yarn package manager:

yarn add nepalipatrojs

Then import:

import { NepaliDatePicker } from 'nepalipatrojs';

CDN

Use directly in HTML via CDN:

<script src="https://cdn.jsdelivr.net/npm/nepalipatrojs/dist/nepali-patro-js.umd.js"></script>

Don't forget the CSS:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/nepalipatrojs/dist/nepali-patro-js.css">

import NepaliPatro from 'nepalipatrojs';

// Convert AD to BS
const adDate = new Date(2024, 0, 1); // January 1, 2024
const bsDate = NepaliPatro.convertADToBS(adDate);
console.log(bsDate); // { year: 2080, month: 9, day: 17 }

// Convert BS to AD
const bsDate2 = { year: 2081, month: 1, day: 15 };
const adDate2 = NepaliPatro.convertBSToAD(bsDate2);
console.log(adDate2); // Date object

// Using array format
const bsFromArray = NepaliPatro.convertADToBS([2024, 1, 1]);
const adFromArray = NepaliPatro.convertBSToAD([2081, 1, 15]);

// Get current Nepali date
const today = NepaliPatro.getCurrentBSDate();
console.log(today); // { year: 2081, month: 4, day: 20 }

// Alternative method
const now = NepaliPatro.now();
console.log(now);
import NepaliPatro from 'nepalipatrojs';

const bsDate = { year: 2081, month: 1, day: 15 };

// Default format (YYYY-MM-DD)
const defaultFormat = NepaliPatro.formatBSDate(bsDate);
console.log(defaultFormat); // "2081-01-15"

// Custom formats
const customFormat1 = NepaliPatro.formatBSDate(bsDate, 'DD/MM/YYYY');
console.log(customFormat1); // "15/01/2081"

const customFormat2 = NepaliPatro.formatBSDate(bsDate, 'MMMM D, YYYY');
console.log(customFormat2); // "Baisakh 15, 2081"

// Nepali language
const nepaliFormat = NepaliPatro.formatBSDate(bsDate, 'MMMM D, YYYY', {
  language: 'np'
});
console.log(nepaliFormat); // "बैशाख १५, २०८१"

// Different format patterns
const patterns = [
  'YYYY-MM-DD',    // 2081-01-15
  'YY/M/D',        // 81/1/15
  'MMMM DD, YYYY', // Baisakh 15, 2081
  'DD MMMM YYYY'   // 15 Baisakh 2081
];
import NepaliPatro from 'nepalipatrojs';

// Get days in a specific month
const daysInBaisakh = NepaliPatro.getDaysInMonth(2081, 1);
console.log(daysInBaisakh); // 31

const daysInFalgun = NepaliPatro.getDaysInMonth(2080, 11);
console.log(daysInFalgun); // 29 or 30 (depends on year)

// Convert to Nepali digits
const nepaliDigits = NepaliPatro.toNepaliDigits(2081);
console.log(nepaliDigits); // "२०८१"

const nepaliYear = NepaliPatro.toNepaliDigits('2081-01-15');
console.log(nepaliYear); // "२०८१-०१-१५"

// Working with month names
const monthNames = {
  en: ['Baisakh', 'Jestha', 'Ashadh', /* ... */],
  np: ['बैशाख', 'जेष्ठ', 'आषाढ', /* ... */]
};

// Validate BS date
function isValidBSDate(year, month, day) {
  try {
    const maxDays = NepaliPatro.getDaysInMonth(year, month);
    return day >= 1 && day <= maxDays;
  } catch (e) {
    return false;
  }
}

API Reference

Complete API documentation for all available methods and options

convertADToBS(date)

Parameters: Date | [year, month, day]
Returns: { year, month, day }
Converts Gregorian date to Bikram Sambat date.

convertBSToAD(bsDate)

Parameters: { year, month, day } | [year, month, day]
Returns: Date
Converts Bikram Sambat date to Gregorian date.

getCurrentBSDate()

Parameters: None
Returns: { year, month, day }
Returns current date in Bikram Sambat calendar.

formatBSDate(date, format, options)

Parameters: bsDate, formatString?, options?
Returns: string
Formats BS date with custom format patterns.

getDaysInMonth(year, month)

Parameters: number, number
Returns: number
Returns total days in specified BS month and year.

NepaliDatePicker

Constructor: new NepaliDatePicker(input, options?)
Options: format, theme, language
Interactive date picker UI component.
